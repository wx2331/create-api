# API自动生成

#### 介绍
基于jdk tools.jar的doc文档生成api说明文档

#### 使用说明
1.引入pom依赖
```
    <dependency>
        <groupId>io.github.wx2331</groupId>
        <artifactId>create-api</artifactId>
        <version>1.0.2</version>
    </dependency>
```
2. 启动项目,用浏览器访问项目的`/io/github/create/api`地址,下载zip文件，解压之后用浏览器打开index.html
